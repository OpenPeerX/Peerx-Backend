# PeerX Project Charter

## Mission
PeerX exists to make peer-to-peer value exchange safe, transparent, and
programmable for everyone.

## Non-goals
- PeerX is not a custodian.
- PeerX is not a derivatives venue for retail.
- PeerX does not provide investment advice.

## Governance
- Changes to this charter require approval from the PeerX maintainers.
- Day-to-day technical decisions are made via pull-request review.
