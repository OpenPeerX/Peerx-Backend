# PeerX Brand Guidelines

## Names
- **Product name**: PeerX
- **Tagline**: Peer-to-peer, simplified.
- **Package name**: `peerx-backend`
- **Container image**: `peerx-backend`
- **Default DB file**: `peerx.db`

## Logo & typography
PeerX uses an upper-camel-case wordmark "PeerX" with a stylised
cross-chain "X" glyph. The wordmark is set in Inter (or a compatible
geometric sans-serif). Never recreate the glyph; use the official SVG.

## Voice
- Confident, technical, neutral.
- Avoid hype words: no "revolutionary", "disruptive", "moon".
- Prefer concrete numbers over claims.
